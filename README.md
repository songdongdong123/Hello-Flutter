    ```
    flutter doctor
    ```

## 使用创建flutter项目

    ```
    flutter create myapp
    ```

## mac下开启模拟器

    ```
    open -a Simulator
    ```

## 运行flutter项目

    ```
    flutter run
    ```

## 命令行下热重载

    在当前命令窗口下按下 r 键

## 安装包

1. 在pubspec.yaml中添加包名

   ```
   flutter资源包查询
   https://pub.dev/flutter/packages
   ```

2. 在命令行中运行：

   ```
   flutter pub get
   ```
